# 📚 Sistema Biblioteca - Exercício API RESTful


Este projeto foi desenvolvido como parte da disciplina **Desenvolvimento de API RESTful**, com o objetivo de **aprender e praticar os fundamentos básicos** na criação de APIs utilizando **Spring Boot**.

---

## 🚀 Objetivos do Exercício
- Compreender a estrutura de um projeto Spring Boot.
- Criar uma entidade `Livro` e implementar operações básicas de CRUD.
- Manipular dados via JSON, simulando o consumo de uma API REST.
- Explorar conceitos de **Controller**, **Service**, **Repository** e **Model**.
- Testar a persistência com banco em memória **H2 Database**.

---

## 🛠️ Tecnologias Utilizadas
- **Java 17**
- **Spring Boot**
- **Spring Web**
- **Spring Data JPA**
- **H2 Database**
- **Maven**

---

## 📌 Funcionalidades Implementadas
- Criar um novo livro (`POST /livros`)
- Listar todos os livros (`GET /livros`)
- Buscar livro por ID (`GET /livros/{id}`)
- Atualizar informações de um livro (`PUT /livros/{id}`)
- Excluir livro (`DELETE /livros/{id}`)
